# Zime Mayeza — Portfolio Website

A modern, futuristic portfolio built with pure HTML, CSS, and JavaScript.

## 🚀 How to publish on GitHub Pages

1. Create a new GitHub repository (e.g. `zime-mayeza-portfolio`)
2. Upload all files, keeping the folder structure:
   ```
   index.html
   css/style.css
   js/canvas.js
   js/main.js
   assets/zime.jpg
   README.md
   ```
3. Go to **Settings → Pages → Source → Deploy from branch → main**
4. Your site will be live at `https://yourusername.github.io/zime-mayeza-portfolio`

## 📁 File Structure

| File | Description |
|------|-------------|
| `index.html` | Main HTML page |
| `css/style.css` | All styles — deep purple & rose gold theme |
| `js/canvas.js` | Animated data analytics background (canvas) |
| `js/main.js` | Interactions: nav, scroll reveal, counters, accordion |
| `assets/zime.jpg` | Profile photo |

## ✨ Features

- Fully mobile responsive
- Animated data analytics canvas background
- Scroll reveal animations
- Interactive experience accordion
- Counter animations on stats
- Cursor glow effect (desktop)
- Deep purple & rose gold colour palette
